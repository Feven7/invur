# invur-bot

Creates a hCaptcha Discord Invite url
(WIP)

# How to use

## Setup Dotenv  

### server/.env

```
TOKEN=
SESSION_SECRET=
DB_HOST=
DB_PASS=
DB_USER=
SECRET=
ID=
REDI_URI=
HCAPTCHA=
PORT=
```

## Start

### Productive

```

 npm run build
 npm run start:prod

```

### Development

```

npm run dev
```
